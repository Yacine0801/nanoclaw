## Google Workspace (via `gws` CLI)

Tu as le CLI `gws` (Google Workspace CLI) disponible via Bash. Compte connecté : `yacine@bestoftours.co.uk`.

### Gmail
- `gws gmail +triage` — résumé inbox non lus (expéditeur, objet, date)
- `gws gmail +read --message-id <ID>` — lire un email complet
- `gws gmail +send --to <email> --subject "..." --body "..."` — envoyer un email
- `gws gmail +reply --message-id <ID> --body "..."` — répondre à un email
- `gws gmail +forward --message-id <ID> --to <email>` — transférer un email
- `gws gmail users messages list --params '{"userId":"me","q":"search query","maxResults":5}'` — rechercher des emails

### Calendar
- `gws calendar events list --params '{"calendarId":"primary","timeMin":"2026-03-19T00:00:00Z","timeMax":"2026-03-20T00:00:00Z","singleEvents":true,"orderBy":"startTime"}'` — lister les événements
- `gws calendar events insert --params '{"calendarId":"primary"}' --json '{"summary":"Meeting","start":{"dateTime":"2026-03-20T14:00:00+01:00"},"end":{"dateTime":"2026-03-20T15:00:00+01:00"}}'` — créer un événement

### Drive
- `gws drive files list --params '{"q":"name contains '\''budget'\''","pageSize":10,"fields":"files(id,name,mimeType,webViewLink)"}'` — chercher des fichiers

### Sheets
- `gws sheets +read --spreadsheet-id <ID>` — lire un spreadsheet
- `gws sheets +append --spreadsheet-id <ID> --range "Sheet1" --json '[["col1","col2"]]'` — ajouter des lignes

### Docs
- `gws docs documents get --params '{"documentId":"<ID>"}'` — lire un document
- `gws docs +write --document-id <ID> --text "contenu"` — écrire dans un document
- `gws docs documents create --json '{"title":"Mon Document"}'` — créer un document

Utilise toujours Bash pour exécuter ces commandes. Parse le JSON pour extraire les infos pertinentes avant de répondre à Yacine.
